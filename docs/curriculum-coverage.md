# 小6・中2 学習指導要領対応表

## 目的

MiraStudy の小6・中2ドリルが、文部科学省の現行学習指導要領にある各内容を扱えているか確認するための対応表です。

ここでの「基礎ドリル実装済み」は、各内容に対して少なくとも1種類の自動出題、採点、3段階ヒント、解説、自動テストがあることを表します。学校の授業そのものを置き換えるものではなく、作図、グラフ作成、証明を書き切る問題、文章題の幅は今後増やします。

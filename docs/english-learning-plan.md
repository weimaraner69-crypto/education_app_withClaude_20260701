# 英語学習の実装計画

## 優先順位

2026年7月14日から、MiraStudyの最優先を算数・数学の拡張から英語学習へ切り替える。
最初の利用者は中学2年生の娘さんを想定する。小6の息子さん向けの英語は、最初の仕組みを
使って次に追加する。

## 学習指導要領との関係

中学校の外国語は、語彙・表現・文法を理解し、聞くこと、読むこと、話すこと、書くことの
活動で活用することを目標とする。MiraStudyは、教科書ごとの進度差に配慮しながら、語彙と
文法を文の中で確かめる短いドリルを最初に作る。

根拠： [中学校学習指導要領（平成29年告示）解説・外国語編](https://www.mext.go.jp/content/20210531-mxt_kyoiku01-100002608_010.pdf)

## 最初の範囲

1. 英単語：英語から日本語、日本語から英語の4択問題。意味だけでなく、短い例文も表示する。
2. 文法：時制、助動詞、比較、不定詞などを、短い文の穴埋め4択で出す。
3. ヒント：文の手がかり、使う語句、答えに近い例文の順で3段階表示する。
4. 学習記録：英語も既存の `attempts` に、教科ID `english` で保存する。

## 今回は作らないこと

- リスニング、発音判定、音声の録音
- AIによるその場での文章問題生成
- スペルの軽微な誤りを自動で正解にする採点

記述式のスペル採点は、許容範囲を保護者が決めてから追加する。最初は選択式に限定し、
誤判定を避ける。
